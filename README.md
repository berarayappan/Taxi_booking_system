# Call Taxi Booking System

## Project Description

This is a console-based Call Taxi Booking System developed using Python. The application allocates taxis to customers based on the nearest available taxi and tracks taxi earnings and booking history.

## Problem Statement

* There are 6 points: A, B, C, D, E, and F arranged linearly.
* Distance between consecutive points is 15 km.
* Travel time between consecutive points is 1 hour.
* All taxis start from point A.
* Minimum fare is Rs.100 for the first 5 km.
* Additional fare is Rs.10 per km after 5 km.
* The nearest available taxi is allocated to the customer.
* If multiple taxis are equally near, the taxi with lower earnings is selected.
* If no taxi is available, the booking is rejected.

## Features

* Taxi allocation based on nearest distance
* Fare calculation
* Booking management
* Earnings tracking
* Booking history maintenance
* Menu-driven console application
